## Meeting For You Backend

## 依赖要求

- Java: JDK 21+

- Maven: 3.8.1+

## 启动指南

1.  普通`mvn`运行方式
   - 检查maven版本: `mvn -v`
   - 导航到项目目录: `cd /grad-back/src`
   - 构建`mvn`项目: `mvn clean install`
   - 启动`spring boot`工程：`mvn spring-boot:run`

2. `mvn warpper`运行方式
   - Unix 系统:  `./mvnw spring-boot:run` 
   - Windows 系统: `mvnw.cmd spring-boot:run`
